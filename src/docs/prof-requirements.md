# Consignes du professeur — Portfolio React/TypeScript

## Objectif
Créer un portfolio personnel propre, structuré, maintenable et déployable.

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui si nécessaire
- react-helmet-async pour le SEO minimal

## Pages obligatoires
- Home
- Projects
- Experience
- Education
- Certifications
- Contact

## Contraintes techniques
- Les données doivent être centralisées dans `src/data/*`
- Routing complet avec `RootLayout`
- Interface cohérente et responsive
- Code clair, maintenable et bien organisé
- Build sans erreur
- Imports corrects
- Typage TypeScript propre

## SEO minimal
Chaque page doit avoir :
- un title
- une meta description

## Attentes de qualité
- pas de duplication inutile
- composants réutilisables
- accessibilité de base
- structure propre
- design cohérent

## Bonus éventuels
- Dark mode
- JSON-LD

## Livraison
- projet fonctionnel
- `npm run build` OK
- `npm run preview` OK
- prêt pour déploiement Vercel
